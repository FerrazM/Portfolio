Habilidades

Python

Python é uma linguagem multiparadigma, o que significa que suporta diferentes estilos de programação, como programação orientada a objetos, programação imperativa e programação funcional. Python é amplamente utilizada em diversas áreas, incluindo ciência de dados, inteligência artificial, desenvolvimento web, automação de tarefas e muito mais.

HTML CSS e JavaScript

HTML, CSS e JavaScript são tecnologias fundamentais para o desenvolvimento de páginas web interativas e atraentes. Embora cada uma tenha uma função específica, elas geralmente são usadas juntas para criar experiências de usuário ricas e dinâmicas.

Django

Django é um framework de desenvolvimento web de alto nível, escrito em Python, que permite criar rapidamente aplicativos web escaláveis e seguros. Ele segue o padrão Model-View-Controller (MVC) e Model-View-Template (MVT), que separa a lógica de negócios do aplicativo da apresentação visual.

Selenium

Selenium é uma ferramenta de automação de testes de software para aplicações web. Ela permite que desenvolvedores e testadores criem scripts automatizados para simular interações de usuário em navegadores web, como clicar em botões, preencher formulários e navegar por páginas. Com isso, é possível testar a funcionalidade e o desempenho de um aplicativo web de forma mais rápida e eficiente do que testes manuais.
